# ICU-AW アプリの配布方法

このフォルダを静的Webサイトとして公開すると、PC・タブレット・スマホから同じURLで利用できます。

## 重要

- 入力データは各端末のブラウザ内に保存されます。
- 端末間ではデータ共有されません。
- URL配信には `https://` が推奨です。
- `file://` で開く場合は、その端末だけで利用できます。

## 配布方法の例

1. この `icu_aw_app` フォルダをWebサーバーに置く
2. `index.html` にアクセスできるURLを配布する
3. 各端末でURLを開いて使用する

## 配置先の例

- 院内・施設内の共有Webサーバー
- GitHub Pages
- Netlify
- Vercel
- レンタルサーバー
- NASのWeb公開機能

スマホやタブレットでは、URLを開いたあとブラウザの「ホーム画面に追加」を使うとアプリのように起動できます。
